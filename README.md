# arch-hexagonal
Project example using hexagonal archtecture based in Arquitetura Hexagonal/Ports and Adapters NA PR√ÅTICA course on Udemy platform.

### Requirements
* Java JDK 17
* Gradle 8.11.1
* Wiremock standalone 3.10.0
* Docker 27.3.1

###### Download Wiremock jars [here](https://wiremock.org/docs/download-and-installation) and put on mock folder.
###### Running Wiremock:
```
$ java -jar wiremock-standalone-3.10.0.jar --port 8082
```
